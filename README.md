# GP Catchment Area Data

This repository contains data related to catchment areas of GP practices in the UK. 

## Data Sources

The primary data sources for this repository are:
1. **GP Catchment Areas data** from ArcGIS Hub, which provides detailed geographical boundaries for GP catchment areas [[1](#references)]. GP practices self-declare their catchment areas annually through the General Practice Annual Electronic Self Declaration (eDEC) process managed by NHS Digital [[2](#references)].

# References

1. GP Catchment Areas data — ArcGIS Hub: https://gp-catchments-data-download-nhsgisscw.hub.arcgis.com/
2. General Practice Annual Electronic Self Declaration (eDEC) — NHS Digital: https://digital.nhs.uk/data-and-information/data-collections-and-data-sets/data-collections/general-practice-annual-electronic-self-declaration-edec
